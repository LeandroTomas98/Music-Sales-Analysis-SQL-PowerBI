# 🎵 Global Music Sales Data Analysis
Analysis of international sales patterns to identify market growth opportunities using **SQL** and **Power BI**.

## 🚀 Overview
This project transforms raw data from the Chinook database into actionable business insights. It demonstrates a complete data workflow:
1. **Extraction:** SQL queries to aggregate revenue by country.
2. **Connectivity:** ODBC Bridge setup to connect SQLite with Power BI.
3. **Visualization:** Interactive dashboard featuring Market Share (Treemap) and Revenue Trends.

## 📈 Key Insights
* Identified **USA** as the primary market with over 40% of top revenue.
* Discovered strong growth potential in **European markets** (France/Germany) for logistics expansion.
* Optimized report performance by pre-filtering data at the database level (SQL).

## 📁 Files
* `SQL Music Industry.sql`: SQL logic for data processing.
* `Sales of music by country.pbix`: Interactive Power BI dashboard.
* `Report_Preview.pdf`: Quick-view document for recruiters without Power BI.
